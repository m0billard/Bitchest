# **How to setup the application**

```
cd Bitchest
```

```
npm i
```

```
composer i
```

## Launch the migrations

```
php artisan migrate
```

## Launch the seeders

```
php artisan db:seed
```

# **How to launch the application**

```
npm run dev
```

```
php artisan serve
```
