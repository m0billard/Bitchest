# **HOW TO SETUP THE PROJECT**

```
 cd Bitchest
```

```
npm i
```

```
composer i
```

## Launch the migrations

```
php artisan migrate
```

## Launch the seeders

```
php artisan db:seed
```

```
php artisan db:seed CryptoCurrencySeeder
```

# **HOW TO LAUNCH THE APPLICATION**

```
npm run dev
```

```
php artisan serve
```
